# 💼 Kit Mini-Services Sécurité

Ton "business en boîte" pour vendre des prestations de cybersécurité simples,
légales, et livrées avec tes propres outils (**DataGuard**, nuclei, etc.).

## Contenu du kit

| Fichier | À quoi ça sert |
|---|---|
| `offre-service.md` | Tes argumentaires de vente prêts à copier (Fiverr, LinkedIn, direct) |
| `process-client.md` | Ta checklist de livraison + modèle d'autorisation écrite |
| `modele-rapport.md` | La structure du rapport que tu remets au client |

## Les 3 services (du + sûr au + avancé)

1. **Scan de secrets dans le code** 🔑 — DataGuard `detectors.py` — 40-80 € — *zéro risque légal, commence par ça*
2. **Analyse anti-phishing + sensibilisation** 🎣 — DataGuard `phishing.py` — 50-100 €
3. **Checkup sécurité de site web** 🌐 — nuclei + vérifs manuelles — 80-150 € — *⚠️ autorisation écrite obligatoire*

## Règle d'or absolue ⚖️

```
On ne teste QUE ce que le client possède ET pour quoi il t'a donné
une AUTORISATION ÉCRITE (voir le modèle dans process-client.md).
Scanner un système sans accord = délit. Pas de zone grise.
```
